Таблица цветов осадков

Цвета осадков

Качественная реакция

Белый творожистый; нерастворимый в воде; нерастворимый в HNO3

Ag+ + Cl- -> AgCl (осадок).
Качественная реакция на соли серебра, качественная реакция на хлорид-ионы.

Белый осадок, нерастворимый в кислотах

Ba2+ + SO4- -> BaSO4 (осадок).
Качественная реакция на соли бария; качественная реакция на сульфат-ионы.

Белый осадок; образуется при добавлении щелочи; растворяется в избытке щелочи

Al3+ + 3OH- -> Al(OH)3 (осадок)
Zn2+ + 2OH- -> Zn(OH)2 (осадок)
При избытке щелочи образуется комплексная соль

Белый осадок; образуется при пропускании газа без цвета и запаха через известковую воду; растворяется при пропускании избытка газа; растворяется в кислотах

Ca(OH)2 + CO2 -> CaCO3 (осадок) + H2O

  • избыток CO2: CaCO3 (осадок) + H2O + CO2 -> Ca(HCO3)2 (гидрокарбонат кальция)
  • растворение в кислотах: CaCO3 + 2H+ -> CO2 + H2O + Ca2+

Осадок светло-желтого цвета (кремового цвета); образуется при приливании AgNO3; нерастворим в HNO3.

Ag+ + Br- -> AgBr (осадок).
Качественная реакция на бромид-ионы; качественная реакция на соли серебра.

Осадок желтого цвета; образуется при приливании AgNO3; нерастворим в HNO3

Ag+ + I- -> AgI (осадок).
Качественная реакция на иодид-ионы; качественная реакция на соли серебра.

Осадок желтого цвета; образуется при приливании AgNO3; растворим в кислотах

3Ag+ + PO33- -> Ag3PO4 (осадок).
Качественная реакция на фосфат-ионы; качественная реакция на соли серебра.

Осадок бурого цвета; образуется при взаимодействии с растворами щелочей

Fe3+ + 3OH- -> Fe(OH)3(осадок).
Качественная реакция на соли железа (III).

Осадок голубого цвета (синего цвета); образуется при взаимодействии с растворами щелочей

Cu2+ + 2OH- -> Cu(OH)2(осадок).
Качественная реакция на соли меди (II).

Осадок черного цвета; образуется при взаимодействии с сульфидами или с H2S

Cu2+ + S2- -> CuS (осадок).
Fe2+ + S2- -> FeS (осадок).
2Ag+ + S2- -> Ag2S (осадок).
Качественная реакция на сульфид-ионы.
